**VirtualCat**
Una app básica de mascota virtual en Android con Jetpack Compose. Puedes alimentar, jugar y ver el estado de tu gato.

**Tecnologías**
Kotlin
Jetpack Compose
ViewModel
Navigation
Funciones
Navegación entre pantallas
Imagen del gato según su estado
Gestión de hambre, felicidad y energía
Virtual Cat App
A virtual pet app for Android built with Jetpack Compose. Take care of your cat by feeding, playing, and monitoring its mood.

**Technologies Used**
Kotlin
Jetpack Compose
Features
Navigation between screens: Home, Food, Game, Settings.
Manages cat state: hunger, happiness, and energy.
Cat image changes depending on its condition.
